================================================================================
                    NITROKEY SECURITY - IMPORTANT INFORMATION
================================================================================

Your Nitrokey uses THREE credentials to protect your cryptographic keys:

1. PIN (Personal Identification Number)
   Default: 123456
   Used for: Decrypting files, signing certificates
   Format: 6-8 characters

2. PUK (PIN Unlock Key)
   Default: 12345678
   Used for: Unlocking PIN after failed attempts
   Format: 6-8 characters

3. Admin Key (Management Key)
   Default: 010203040506070801020304050607080102030405060708 (hex)
   Used for: Generating keys, creating certificates
   Format: 16/24/32 bytes (hex or password)

The following table shows what authentication each operation requires:

Operation              | PIN | PUK | MGM | Remarks
-----------------------|-----|-----|-----|----------------------------------
Change PIN             |  x  |     |     |
Change PUK             |     |  x  |     |
Change MGM (Admin Key) |     |     |  x  |
Unblock PIN            |     |  x  |     |
Generate key           |  x  |     |  x  |
Read certificate       |     |     |     | Does not require authentication
Write certificate      |  x  |     |  x  |

Legend:
  PIN = Personal Identification Number
  PUK = PIN Unlock Key
  MGM = Management Key (Admin Key)
  x   = Authentication required

RECOMMENDED FIRST STEPS:
  1. Change admin key:  nise --set-admin-key  (DO THIS FIRST!)
  2. Change PIN:        nise --set-pin
  3. Change PUK:        nise --set-puk
  4. Initialize slot:   nise -i -s 85
